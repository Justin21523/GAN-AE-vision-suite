# Demo Artifacts

This folder contains small, safe-to-commit artifacts for portfolio screenshots and static demos.

- `demo_scenario.md`: the recommended interviewer walkthrough.
- `sample_run_metrics.json`: representative metrics shown by the UI overview.
- `sample_data_report.json`: compact data-quality report shape matching `src.scripts.data_report`.

The full local demo still produces real artifacts under ignored directories such as `logs/`, `outputs/`, and `.ai_cache/`.
