What it demonstrates

End-to-end ML engineering: reproducible configs, dataset checks, model training, checkpoint inference, metrics, artifact browsing and a UI that makes the work understandable.

How to view it

The public site is static for safety. The full demo runs locally with scripts/dev_fullstack.sh, then opens the React dashboard at 127.0.0.1:5173.

Interview flow

Start on Overview, run Data Report, launch a CPU smoke training job, inspect Runs, then load a checkpoint in Sampler to generate a deterministic image grid.
